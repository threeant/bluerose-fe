# MUSIC REQ APP 

## Quick Start

### Installation

``` bash
$ npm install
```

or

``` bash
$ yarn install
```

prettier 세팅



### Run

``` bash
# dev server with hot reload at http://localhost:3000
$ npm run start 
```

or 

``` bash
# dev server with hot reload at http://localhost:3000
$ yarn run start
```
