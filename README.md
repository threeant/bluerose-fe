# MUSIC REQ APP 

### Installation

``` bash
$ npm install
```

or

``` bash
$ yarn install
```


### Run

``` bash
# dev server with hot reload at http://localhost:3000
$ npm run start 
```

or 

``` bash
# dev server with hot reload at http://localhost:3000
$ yarn run start
```
